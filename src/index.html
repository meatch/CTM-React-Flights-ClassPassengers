Flights Class Passenger Modal :: React

Requirements/Todo

  1. Adults (18-64)
  2. Seniors (65+)
  3. Children (2-17), must provide ages as well
  4. Default selection: 1 Adult, Economy
  5. Selecting values in popover modal for passenger selections will update the output value. When the user selects multiple passengers of a single passenger type, the input field will display that passenger type (e.g., "2 Seniors"). If and only if the user selects passengers of different types will the input read "(X) Passengers".
  6. User can select up to 8 passengers, with a minimum of 1 adult or senior.
  7. For all popovers: Clicking outside or clicking "Continue" saves the current selections. Clicking "X" acts as a cancel and reverts to the last saved selections.
